## 🔷 Project Name: Digital Tools

## 📌 Description

Digital Tools is a modern React-based web application that allows users to explore, select, and purchase various digital services or tools through an interactive interface. The platform simulates a subscription-based marketplace where users can browse different digital products, view their features, and manage selected items using a dynamic cart system. It focuses on providing a smooth user experience with real-time interactions, clean UI, and efficient state management.

---

## ⚙️ Technologies Used

* **React.js** – Component-based UI and state management
* **JavaScript (ES6+)**
* **Tailwind CSS** – For styling and responsive design
* **React Toastify** – For notifications
* **Vite** – Fast development and build tool

---

## ✨ Key Features

### 🛒 Dynamic Cart System

Users can add products to the cart, prevent duplicate entries, remove items, and see the total price calculated using `reduce()`.

### 🔄 Tab-Based View Switching

Seamless switching between **Products** and **Cart** using React state (`useState`) without page reload.

### ⚡ Real-Time Feedback & Validation

Integrated toast notifications for actions like adding items, removing items, duplicate warnings, and successful checkout.

### 📦 Product Data Handling (Async)

Products are fetched from a local JSON file and rendered dynamically.
